# Hack The Box CTF Writeups

This repository contains my Hack The Box challenge and CTF writeups, documenting my methodology, technical notes, tools used, and lessons learned from hands-on cybersecurity labs.

The goal of this repository is to demonstrate practical security analysis skills through structured writeups across topics such as enumeration, web exploitation, cryptography, forensics, reverse engineering, binary exploitation, networking, and industrial control system security.

## Purpose

This repository is meant to show:

- A consistent methodology for approaching CTF-style security challenges
- Hands-on experience with common offensive and defensive security tools
- Technical documentation skills
- Problem-solving process, not just final answers
- Growth across multiple cybersecurity domains

## Topics Covered

- Web exploitation

- Windows privilege escalation
- Cryptography
- Forensics

- Industrial control systems
- BACnet and building automation security
- Operational technology security concepts
